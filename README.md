# 風神

Rendering engine. For fun. C++20.

## Building

Requires:
- CMake >= 3.21.0
- C++ compiler >= C++20
- C compiler

```bash
git clone --recursive git@github.com:heyallnorahere/fuujin.git
cd fuujin

cmake --preset default
cmake --build --preset default
```

I like [Anji Mito](https://www.dustloop.com/w/GGST/Anji_Mito#Fuujin).
